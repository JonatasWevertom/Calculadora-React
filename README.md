# Primeiros passos com "Create-React-App"

Este projeto foi inicializado com [Create React App] (https://github.com/facebook/create-react-app).

## Scripts Disponíveis

### Primeiro passo voce deve dar um `git clone` para clonar o projeto, em seguida você deve dar um` npm install` para instalar as dependências, e por fim deve dar um `npm start` para rodar o projeto em seu navegador .

##

Executa o aplicativo no modo de desenvolvimento. \
Abra [http: // localhost: 3000] (http: // localhost: 3000) para visualizá-lo no navegador.

A página será recarregada se você fizer edições. \
Você também verá quaisquer erros de lint no console.
